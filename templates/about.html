What is This?

The Warhammer 40k Cheat Sheet Generator converts BattleScribe army list exports into clean, printable reference sheets for tabletop gaming. No more flipping through codexes during games!

Features

  • ✅ Parse BattleScribe text exports
  • ✅ Automatic faction detection
  • ✅ Auto-download catalogues from BSData GitHub
  • ✅ Complete unit stats and weapon profiles
  • ✅ Abilities organized by game phase
  • ✅ Faction abilities automatically detected
  • ✅ Print-optimized layout
  • ✅ Support for enhancements and warlords
  • ✅ Multiple output formats (HTML, Markdown)

Supported Factions

All factions are supported! The app automatically downloads catalogues from the official BSData repository:

  • Chaos: Death Guard, Thousand Sons, World Eaters, Chaos Space Marines, Chaos Daemons, Chaos Knights, Emperor's Children
  • Imperium: Space Marines (all chapters), Astra Militarum, Adepta Sororitas, Adeptus Custodes, Grey Knights, Imperial Knights, Deathwatch, and more
  • Xenos: Necrons, Orks, Tyranids, T'au Empire, Aeldari, Drukhari, Leagues of Votann, Genestealer Cults

Catalogues are fetched directly from BSData/wh40k-10e and cached locally.

How It Works

  1. You export your army list from BattleScribe as text
  2. The app automatically detects your faction
  3. If needed, it downloads the latest catalogue from BSData GitHub
  4. The parser extracts unit names, wargear, and enhancements
  5. The generator looks up complete datasheets from faction catalogues
  6. Everything is formatted into a clean, organized cheat sheet

Privacy & Data

Your army lists are processed in real-time and not stored on our servers. All processing happens when you click "Generate" and the data is immediately discarded afterward.

Tips for Best Results

  • Use the latest BattleScribe catalogues when building your army
  • Export as Text, not HTML or other formats
  • Copy the entire army list including headers and faction name
  • The first few lines should include your faction name (e.g., "Death Guard", "Space Marines")
  • Use HTML format for printing, Markdown for editing
